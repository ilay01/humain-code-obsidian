Nova Assistant v2.1

A complete, honest look at what this AI can do, how it was built, and where it falls short. Read before you trust.

01 — What it does

What goes in, and what comes out

Nova Assistant accepts text, images, and PDF documents as input. It generates text responses including summaries, code, analysis, and conversational replies.

  • Text input ✓ Supported
  • Image understanding ✓ Supported
  • PDF parsing ✓ Supported
  • Code generation ✓ Supported
  • Real-time web data ✗ Not available
  • Audio processing ✗ Not available
  • Video understanding ✗ Not available
02 — Honest limitations

What it cannot do — and might get wrong

Nova's training data has a hard cutoff at September 2024. It has no knowledge of events after this date, and cannot access live information.

⚠️ Nova may generate plausible-sounding but entirely fabricated citations, statistics, and quotes. Always verify critical information independently.

It is explicitly not designed for medical diagnoses, legal counsel, or financial advice. Outputs in these domains should be treated as informational only.

03 — How it was built

Under the hood

Nova is a decoder-only transformer with 70 billion parameters, fine-tuned using reinforcement learning from human feedback (RLHF) and constitutional AI methods.

ArchitectureTransformer (decoder-only)
Parameters70 billion
Context window128,000 tokens
Training data size~2.4 trillion tokens
Data sourcesWeb crawl, books, code repos, licensed data
Languages13 (English primary)
